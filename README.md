### Prerequisites

Required things & version
* PHP 7.2.5+
* Nodejs 12.16.1+
* MySQL 8.0.20+

### Installation

1. Clone the repo
```sh
git clone https://github.com/Team-SyntaxErrors/LaravelReact-E-Commerce.git
```
2. Install Composer in project
```sh
composer install
```
3. Set Database name to "grocerydb"
